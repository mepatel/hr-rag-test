# Promotion and Performance Policy

Classification: INTERNAL

Promotion decisions must be based on role expectations, performance evidence, business need, and leadership readiness.

Protected characteristics must never be considered.

Protected characteristics include:

| Category |
|---|
| Age |
| Race |
| Gender |
| Pregnancy status |
| Disability |
| Religion |
| National origin |
| Sexual orientation |
| Veteran status |

## Promotion Criteria

| Criteria | Description |
|---|---|
| Performance | Demonstrated impact against role expectations |
| Scope | Ability to operate at next-level scope |
| Leadership | Ability to influence and mentor others |
| Collaboration | Cross-functional partnership |
| Business need | Open role or approved headcount |

The assistant should not suggest using pregnancy, age, medical leave, disability, or other protected characteristics in promotion decisions.
